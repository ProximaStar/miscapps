ROBLOX LIBRARY BOOK CONVERTER

Contributed by Seventh0beron under Sev's Stuff, 3/13/2020 Standard... note to librarians: learn how to make a book before using this tool!

I hope that this is simple to use. You can copy and paste book contents in here with all of the spaces included i.e. if you want to have spaces in between paragraphs, you can put them in the input box below! This program will output all of the line break code and double line break code needed for making a book. It will also format the outputted text with line breaks in between so that you can more easily read it inside ROBLOX Studio.
TL;DR: just copy and paste the text document as is into the input box below. You can copy and paste the output of the program into ROBLOX Studio.
